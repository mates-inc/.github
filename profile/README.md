</div>

### 株式会社メイツとは？

Web プラットフォームのアプリケーションを主軸とした日本の EdTech 企業です．\
学習塾向け ICT 教材 [aim@](https://mates-app.jp/aim/) や，塾管理システム **reco** を始めとしたアプリケーションの開発・運用や研究開発，**学習塾の運営** を行なっています．\
ほぼ全てのシステムで Vue.js / TypeScript / AWS を使用しており、Vue や AWS はもちろん，様々なコミュニティでの活動や OSS 活動をしているメンバーも多数在籍しており，チーム全体で OSS への参加が前向きな組織です．

[会社 HP](https://mates-edu.co.jp/)

(エンジニアリングチームの HP やブログ等は現在準備中です，)

## 組織

株式会社メイツは塾運営事業とシステム開発の 2 つに別れ，この GitHub Orgnaization は後者を担当する「エンジニアリングチーム」のものです．

エンジニアリングチームには以下のポストがあります．

- ディレクター (部長)
- チーフエンジニア ([@go-to-k](https://github.com/go-to-k), [@ubugeeei](https://github.com/ubugeeei))
- プロジェクトマネージャ
- リードエンジニア (フロントエンド，バックエンド，インフラ)
- エンジニア (フロントエンド，バックエンド，インフラ)

## 求人

現在，株式会社メイツではリードエンジニアを募集しています．

チーフエンジニアと一緒に一緒にプロジェクトを牽引してくれるような，高い技術力を持つエンジニアを探しています！
